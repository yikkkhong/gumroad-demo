IMAGE SPECIFICATIONS:
---------------------
- Format: JPG or PNG
- Project Images: 800x1200px (2:3 portrait ratio) recommended - VERTICAL ORIENTATION
- About Image: 800x800px (square) recommended
- File Size: Keep under 500KB for optimal loading
- Quality: 80-90% JPEG quality is ideal

IMPORTANT: Photos should be in PORTRAIT (vertical) orientation for the MÃ¶bius orbit effect!

OPTIMIZATION TIPS:
------------------
1. Use tools like TinyPNG.com or Squoosh.app to compress images
2. Ensure images are properly cropped before uploading
3. Use consistent dimensions for a professional look
4. Consider using WebP format for better compression (rename to .webp)

PLACEHOLDER IMAGES:
-------------------
If you don't have images yet, you can use placeholder services:
- https://picsum.photos/800/1200 (random portrait images)
- https://placeholder.com/800x1200 (solid color placeholders)
- https://unsplash.com (free high-quality stock photos - use portrait orientation)

Simply download and rename them according to the naming convention above.
Remember to use PORTRAIT/VERTICAL orientation (taller than wide)!
