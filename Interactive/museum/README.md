
# Virtual Exhibition Portfolio Display

An interactive, single-page web project that showcases a small virtual portfolio exhibition experience. This template contains the static website (HTML, CSS, and JavaScript) used for the interactive Virtual Exhibition Portfolio Display demo.

**Project status**: Prototype / demo

**Tech**: Vanilla HTML, CSS, and JavaScript (no build step required)

**Features**
- **Interactive exhibits**: Visitor interactions handled in `script.js`.
- **Responsive layout**: Styling in `style.css` adapts to common screen sizes.
- **Zero-dependency**: No frameworks or bundlers required.

**Getting Started**
Prerequisites: A modern web browser (Chrome, Edge, Firefox, Safari).

**File Structure**
- `index.html`: Main HTML file.
- `script.js`: Interactive behavior and app logic.
- `style.css`: Visual styles and responsive layout.
- `README.md`: This file.

**Development Notes**
- Editing: Modify `script.js` for behavior and `style.css` for visuals.
- No build tools: Changes are reflected immediately when you refresh the page.
- Debugging: Use the browser DevTools console for runtime errors and logs.
