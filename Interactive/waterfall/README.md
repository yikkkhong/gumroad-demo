# Waterfall Portfolio - The Flow

A physics-based interactive portfolio template featuring a fluid particle simulation where water droplets interact with HTML content blocks.

## Project Structure

- **index.html** – Main structure, text, navigation, layout  
- **style.css** – Styling, cursor, glassmorphism, spacing  
- **script.js** – Particle simulation, physics, collisions, mouse interactions

## How to Run

1. Download `index.html`, `style.css`, and `script.js` into the same folder.  
2. Open `index.html` in any modern web browser.
