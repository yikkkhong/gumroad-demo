# Law Firm Management System

Law Firm Management System is a comprehensive web-based management system designed to streamline case management, client information, appointments and billing in a single, easy-to-use platform.

## Features

- **Dashboard**: Overview of cases, appointments, and important metrics
- **Case Management**: Track all case details, status, and related documents
- **Client Directory**: Manage client information and communication history
- **Calendar**: Schedule and manage appointments, court dates, and deadlines
- **Billing & Invoices**: Generate and track invoices and payments
- **Document Management**: Store and organize legal documents
- **User Management**: Role-based access control for staff members

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Styling**: Custom CSS with CSS Variables for theming
- **Icons**: Font Awesome
- **No Backend Required**: All data is stored in the browser's memory (for demo purposes)

## Getting Started

1. Clone the repository:
   ```bash
   git clone [your-repository-url]
   ```
2. Open `index.html` in a modern web browser
3. No installation or database setup required for the demo version

## Usage

1. **Navigation**: Use the sidebar to switch between different sections
2. **Adding Data**: Click the "+" buttons to add new cases, clients, or events
3. **Calendar**: View and schedule appointments using the interactive calendar
4. **Search**: Use the search functionality to quickly find cases or clients
