## ğŸ¨ Screenshot Best Practices

### Preparation
1. **Use impressive data**
   - Professional job titles
   - Well-known companies
   - Impressive metrics
   - Real-looking projects

2. **Clean browser**
   - Hide bookmarks bar
   - Close unnecessary tabs
   - Use clean browser profile
   - Remove extensions

3. **Optimal settings**
   - 100% zoom level
   - Full HD resolution (1920x1080)
   - Good lighting (for photos)
   - High-quality display

### Capture Tools

**For Desktop:**
- **Windows:** Snipping Tool, ShareX, Greenshot
- **Mac:** Screenshot (Cmd+Shift+4), CleanShot X
- **Cross-platform:** Lightshot, Nimbus Screenshot

**For Mobile:**
- **iOS:** Native screenshot + Mockup generator
- **Android:** Native screenshot + Device frame
- **Browser DevTools:** Chrome/Firefox responsive mode

**For GIFs:**
- **ScreenToGif** (Windows)
- **Kap** (Mac)
- **LICEcap** (Cross-platform)
- **Gifox** (Mac)

### Editing

**Basic Edits:**
- Crop to remove unnecessary space
- Add subtle drop shadow
- Slight brightness/contrast adjustment
- Compress for web (keep under 500KB)

**Advanced Edits:**
- Add device frames (for mobile)
- Add annotations/highlights
- Create comparison images
- Add gradient backgrounds

**Tools:**
- **Figma** - Professional mockups
- **Photoshop** - Advanced editing
- **Canva** - Quick edits
- **Photopea** - Free online editor
